querysrting 모듈
	쿼리스트링 형태의 문자열 -> 객체의 형태로 변환(.parse)
	
	객체 형태의 값 -> 쿼리스트링 형태(.stringify())
	
url 모듈 - url 관련된 기능
	.parse -> URL 을 구성 요소로 분해
	.format -> 분해된 URL을 결합
	
	

url 모듈의 URL 객체
	URL - class -> 생성자
	
	const url = newURL("URL 경로");
	
	- 쿼리스트링의 가공 편하게 해주는 편의 기능
	- URL인스턴스.searchParams -> 쿼리스트링 관리 객체
			Object 객체
				.get("속성명") -> 조회
				.getAll() -> 전체 조회
				.append("속성", "값") -> 쿼리스트링 값 추가
				.delete("속성") -> 쿼리스트링 값 삭제
				.set("속성", "값") -> 쿼리스트링 추가 또는 변경
				.toString() -> 쿼리스트링 관리 객체 -> 쿼리스트링 문자열로 변환
				(객체, 인스턴스를 출력할 때도 호출)
